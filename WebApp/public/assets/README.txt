Isle Overlay — web/assets/
================================

This folder holds the Gateway basemap image that the overlay renders under
the POI markers + player dot.

-----------------------------------------------
gateway.png — PROCEDURALLY GENERATED, not user-supplied
-----------------------------------------------

    gateway.png   7800 x 7800 px, ~2 MB, PNG

As of 2026-04-16 this file is auto-generated by

    scripts/generate_gateway_map.py

from the POI coordinate data in web/pois.json and the calibration in
config_default.json. No copyrighted game imagery, no Vulnona webp tiles,
no tracing — every pixel is drawn programmatically from public POI facts.

You do NOT need to drop your own gateway.png here anymore. The checked-in
PNG is the authoritative basemap.

-----------------------------------------------
Regenerating gateway.png
-----------------------------------------------

Whenever POI data or calibration changes (e.g. a new VulnonaMAP import, a
new Gateway revision), regenerate the basemap:

    python scripts\generate_gateway_map.py

or from the repo root with the venv activated. Runtime ~13 seconds on a
typical laptop. The --seed flag (default 20260416) makes output fully
deterministic — same seed + same POI data = byte-identical PNG.

Useful flags:

    --size 4096          Smaller output (faster, lower quality)
    --seed 12345         Different noise pattern
    --dry-run            Print classification summary without writing PNG
    --quiet              Suppress progress output

Dependencies: Pillow + NumPy (listed in requirements.txt as optional,
script-only deps — the overlay itself doesn't need them at runtime).

-----------------------------------------------
OPTIONAL:
-----------------------------------------------

    icon.ico    Windows icon used by the PyInstaller build (any
                standard .ico, 256x256 recommended with multiple
                sizes). If absent, the exe uses the PyInstaller
